## 1.0.0 - 2025-11-14
### New
- Initial release of OPOSS Vertiv/Liebert UPS monitoring plugin for Checkmk
- SNMP-based monitoring with 52+ OIDs tracked
- Three monitoring services: Battery, Power, and Environment
- Intelligent fallback from Liebert enterprise OIDs to RFC1628 standard OIDs
- Comprehensive alarm monitoring (9 alarm conditions)
- Regional configuration support for voltage and frequency thresholds
- Rich graphing with metrics, graphs, and perfometers
